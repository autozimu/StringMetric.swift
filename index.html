StringMetric.swift

String metric functions implemented in Swift.

Currently, following functions are implemented, - distance: alias of distanceLevenshtein - distanceLevenshtein: Levenshtein distance - distanceDamerauLevenshtein: Damerau-Levenshtein distance - distanceHamming: Hamming distance - distanceMostFreqK: Most frequent k characters - distanceJaroWinkler: Jaro-Winkler distance

Usage

For Swift Package Manager users, add this module as dependency in Package.swift,

.Package(url: "https://github.com/autozimu/StringMetric.swift.git", majorVersion: 0)

Example

"kitten".distance(between: "sitting")    // => 3
"君子和而不同".distance(between: "小人同而不和")    // => 4
